FRAMEWORKS WEB

¿Qué es un framework Web?

Los frameworks web son un conjunto de herramientas, estilos y librerías dispuestas a través de una estructura o esqueleto base, para el desarrollo de aplicaciones web más escalables y sencillas de mantener. En esta página web vamos a hablar de los 3 frameworks modernos mas usados en la actulidad, todos tienen sus similutudes y sus difencias.

ReactJS

Este es el framework web moderno mas usado actualmente (en occidente), grandes empresas lo usan, desde WhatsApp hasta Uber pasando por su creador Facebook

VUEJS

Este framework es uno de los mas usados, el repositorio oficial de github se situa en tercera posición respecto a la cantidad de "Likes" recividos dada su gran influencia en China ya que el creador de esta es Chino. Gitlab o Nintendo son alguna de las empresas que usan este framework

ANGULAR

Angular es propiedad de google y portanto su popularidad es bastante alta, este framework a diferencia del resto usa como lenguaje TypeScript. Empresas como Paypal, Microsoft y por supuesto google lo usan en sus apliaciones

React

A continuación vamos a hablar en más profundidad de un de los frameworks, en este caso React, este framework se basa en crear componentes lo más reusable posible y poder manipularlos en tiempo real a través de su estado. Esto quiere decir que fácilmente podremos modificar el HTML de nuestra página web, por tanto podemos crear una aplicación web en una sola página de manera muy fácil. Esto lo consigue a través del lenguaje JSX y su virtual DOM.

El DOM es la parte del navegador donde se renderiza el HTML, es decir donde se "incrustan" el HTML. Los creadores de React pensaron una gran idea para poder cambiar los elementos del DOM sin tener que refrescar todo el DOM cada vez que detecte algún cambio. Es tan siempre como tener un DOM virtual, es decir un DOM que nunca se va a pintar, pero es una réplica del que estamos viendo en el navegador, con esto conseguimos que si actualizamos cualquier contenido en el HTML, como por ejemplo cambios en la base de datos, nuestro DOM virtual lo cambiara y ahora comparará su versión con el DOM del navegador y calculara los cambios necesarios para mostrar el nuevo contenido actualizando lo menos posible.

El JSX es un lenguaje que usa react que parace HTML pero en realidad es JavaScript, con esto conseguimos que con el propio JavaScript poder editar el html. Vamos a hacer una prueba para ver que cuando exportamos una pagina en React y detecta este lenguaje, JSX, nos va devolver JavaScript.
